Erhebung und Verarbeitung personenbezogener Daten

Die Extension "Hidden GPT" dient ausschließlich dem Zweck, automatisierte Antworten auf Webseiten mithilfe der OpenAI API bereitzustellen. Dabei werden nur folgende Daten verarbeitet:

  • Eingegebene Informationen: OpenAI API-Schlüssel und Konfigurationen wie Modelleinstellungen und benutzerdefinierte Texte. Diese Daten werden lokal oder synchron gespeichert.
  • Webseiteninhalte: Inhalte der aktiven Webseite (z. B. Fragen und Antwortoptionen), um diese an die OpenAI API zu senden.
  • Nutzeraktivitäten: Benutzerdefinierte Tastenkombinationen werden verarbeitet, um die Funktionalität zu aktivieren.

Datenweitergabe an Dritte

Die Extension gibt Daten ausschließlich in folgendem Fall an Dritte weiter:

  • OpenAI API: Inhalte der Webseite und API-Konfigurationen werden über HTTPS an die OpenAI API gesendet. Die Datenverarbeitung erfolgt gemäß den Datenschutzbestimmungen von OpenAI (OpenAI-Datenschutzerklärung).

Es erfolgt keine Weitergabe an andere Dritte oder zu kommerziellen Zwecken.

Speicherung von Daten

Folgende Daten werden in der Chrome-Synchronisation oder lokal gespeichert:

  • OpenAI API-Schlüssel
  • Modelleinstellungen
  • Zusätzliche Texte
  • Benutzerdefinierte Tastenkombinationen

Es werden keine Daten außerhalb des Geräts oder der Chrome-Umgebung gespeichert.

Rechtsgrundlage der Datenverarbeitung

Die Verarbeitung der Daten erfolgt auf Grundlage von:

  • Art. 6 Abs. 1 lit. a DSGVO: Einwilligung, da der Nutzer die Daten aktiv eingibt.
